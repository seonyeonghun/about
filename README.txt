1.  HTML , CSS, Javascript(jQuery) ==> View Part [뷰 파트] / 1주간

   - HTML5 : 파일이름.html   [HTML]
    > 마크업 언어(Markup Language)
       의미를 부여, 설명하는 듯한 언어
      <title>한울직업전문학교</title> : 제목
      <h1>인사말</h1> : 큰 제목
      ※ h1~h6까지 명령어가 존재함. title은 문서의 제목, h1~h6은 내용중 구별되는 제목
      <p>내용 텍스트</p> : 길거나 짧은 텍스트를 작성하는 명령어, 문장 태그
      <img src="이미지 출처/형식" alt="이미지 설명"> : 이미지 삽입 태그
      <a href="#">클릭 텍스트</a> : 클릭 태그(anchor), 클릭하는 텍스트 표시
   
   > 자동완성 기능
   1) 새 파일 만들기 : index.html
   2) ! + Tab / Enter : 알아서 12줄의 기본 태그라인 추가해줌
   3) lang="en" : ko로 바꾸면 '번역할까요?' 표시 안뜸
   4) 화면에 표시하는 내용은 <body>..</body>에 작성
   
   - CSS3 : 파일이름.css
   - Javascript [jQuery] : 파일이름.js
  ※ 구글 > visual studio code or VSC 입력, MS 다운로드 링크!
